  ## Case Studies

  ### EcoCart — Sustainable Shopping Companion
  A mobile app that helps consumers make sustainable shopping choices through barcode scanning, sustainability scores, and personalized discounts from partner stores.
  - 78% positive user feedback on the business model
  - 150+ user interviews conducted for validation
  - Built with React Native, Node.js, MongoDB

  ### Disaster Defender — Peace Foundation Charity Game
  A video game that gamifies disaster relief experiences while enabling real-time charitable donations to humanitarian causes.
  - Designed for 1K+ real-time charity donations
  - Coordinated 3 cross-functional teams with 12 members
  - Built with Unity, Blender, C#, STEAM SDK

  ### Alcan Dental — AI-Powered Staff Training Platform
  A gamified training platform for dental front desk staff featuring AI-generated interactive calls, educational media, and a RAG-powered chatbot.
  - Delivered AI chatbot with FastAPI and Pinecone RAG architecture
  - Created 19 Figma files with 15 low-fidelity user flows
  - Led cross-functional team of 12+ members

  ## Getting Started

  ```bash
  # Install dependencies
  npm install

  # Start the development server
  npm run dev
  ```

  The app runs on port 5000 with both frontend and backend served together.
